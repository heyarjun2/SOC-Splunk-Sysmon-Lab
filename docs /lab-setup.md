# Lab Setup

## Goal
Create a small SOC-style detection lab using Splunk and Sysmon.

## Planned Environment
- 1 Windows machine or VM
- Sysmon installed for process and event monitoring
- Splunk installed or accessible
- Splunk Universal Forwarder for sending logs

## What will be configured
- Windows Event Log collection
- Sysmon event collection
- Splunk index and data ingestion
- Test activity generation for detections

## Expected Output
- Searchable logs in Splunk
- Detection queries
- Screenshots of alerts and dashboards
